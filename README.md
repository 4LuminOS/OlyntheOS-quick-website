# OlyntheOS Website (beta) 🌐

This repository contains the source code for the official **LuminOS** home page

This is just a preview: the host, domain, and website will slightly change as LuminOS gets... better.

## 🚀 Live Site

Visit the website here: **https://4luminos.github.io/OlyntheOS-quick-website**

## 🛠️ About

This is a lightweight, responsive showcase website built with:
* **HTML5**
* **CSS3** (Modern Bento Grid kind of design)
* **No external frameworks** (Clean & Fast)
